# Vera viewer

## Dependicies

* NodeJS v6.x.x
* Electron v1.6.11

## Building

* Install packages

```bash
cd <repo>
npm install
```

* Running app

```bash
electron .
```

* Building packages

```bash
npm run package-linux  # Linux package
npm run package-win  # Windows package
npm run package-mac  # MacOS package
```
